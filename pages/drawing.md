# Drawing as a way of looking

Drawing is less about recording an object than deciding which relationships matter. A line can describe an edge, a direction, a duration, or simply the pressure of a hand.

## A short practice

1. Pick an object you can hold.
2. Draw its negative space without outlining the object.
3. Turn the page and redraw it with one continuous line.
4. Compare decisions, not accuracy.
